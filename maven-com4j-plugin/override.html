com4j

Maven2 Plugin for Java->COM Bridge using com4j. This plugin will create .java files for a Microsoft COM interface on Windows systems.

This maven plugin by default uses a fixed version of com4j (which is normally the latest when the plugin was released.) To use a newer version of com4j (for example perhaps you'd need to do this to pick up a bug fix), change your POM and add the following text:

      <plugin>
        <groupId>org.jvnet.com4j</groupId>
        <artifactId>maven-com4j-plugin</artifactId>
        <executions>
          ...
        </executions>
        <dependencies>
          <dependency>
            <groupId>org.jvnet.com4j</groupId>
            <artifactId>tlbimp</artifactId>
            <version>20080105</version>  <!-- version of your choice -->
          </dependency>
        </dependencies>
      </plugin>
